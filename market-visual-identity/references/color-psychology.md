# Color Psychology by Industry — Reference

## How to use this file
Load this reference in Step 3 of market-visual-identity. For the user's sector, find the matching entry. Use the dominant palette as basis for Option A ("Autoridad/Confiable"), then derive Options B and C by shifting hue, saturation, or using an alternative from the "Alternative directions" column.

**Key rule**: Never apply generic colors. Every color choice must be justified by the sector's psychological needs and audience expectations.
