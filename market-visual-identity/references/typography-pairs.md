# Typography Pairs Reference

## How to Use

Match the brand's personality profile to one of the 7 profiles below. Select the pair (A, B, or C) that best fits the sector and tone. Each pair includes Google Fonts URLs, recommended weights, and modular scale. Apply the scale tables in the "Modular Scales" section to generate the full type ramp.

## Modular Scales

Selecciona la escala según el par elegido. Aplica el multiplicador al tamaño base (16px por defecto).

| Scale | Ratio | Uso típico |
|-------|-------|-----------|
| Minor Third | 1.2 | Interfaces muy densas, dashboards |
| Major Third | 1.25 | Perfiles cálidos, formales moderados |
| Perfect Fourth | 1.333 | Uso general digital, SaaS, tech |
| Perfect Fifth | 1.5 | Lujo, display bold, deportivo |
| Golden Ratio | 1.618 | Lujo máximo, editorial de alto impacto |

## Typography Rules

1. **Max 2 families.** Never combine more than two typeface families in a single brand system.
2. **Max 3 weights.** Use at most 3 weights per family (e.g., 400 / 600 / 700). More weights create visual noise without hierarchy benefit.
3. **Min 16px body.** Body copy must never go below 16px (1rem) on desktop. Mobile minimum: 15px.
4. **Line-height 1.5 for body.** Body text line-height: 1.5. Headings: 1.1–1.3 depending on size (larger headings need tighter leading).
5. **Letter-spacing rules.** Body: 0 to 0.01em. Headings above 40px: −0.02em to −0.03em (tighter). All-caps labels: 0.08em to 0.15em.
6. **Heading font stays out of paragraphs.** The heading/display font is exclusively for h1–h4, hero text, and pull quotes. Never use it for body paragraphs, captions, or UI labels.
