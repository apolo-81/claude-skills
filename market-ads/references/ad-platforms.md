# Ad Platform Reference — Specs, Benchmarks, and Formats

## Google Ads

### Responsive Search Ads (RSA) — Character Limits
- Headlines: up to 15, max 30 characters each
- Descriptions: up to 4, max 90 characters each
- Display URL paths: 2 fields, 15 characters each

### Headline Angles to Cover (minimum 10)
1. Brand name + value prop
2. Pain point + solution
3. Specific benefit + number
4. Social proof headline
5. Urgency / offer
6. Question hook
7. How-to framing
8. Competitor comparison
9. Feature-focused
10. Action-oriented

### Descriptions to Cover (minimum 4)
1. Value proposition + CTA
2. Features + benefits breakdown
3. Social proof + trust signal
4. Urgency + offer details

### Keyword Strategy
- 10–15 high-intent keywords per ad group
- Match type mix: exact, phrase, broad match modified
- Negative keywords list: 10–20 terms to exclude
- 3–5 ad groups organized by theme

### Performance Max Campaigns
- Asset groups organized by audience segment
- Headlines: 15 short + 5 long variations
- Descriptions: 5 variations
- Image specs: 1200×1200 (square), 1200×628 (landscape), 960×1200 (portrait)
- Video assets: 10–30 seconds recommended
- Audience signals: custom segments, in-market, affinity

### Format Specs

| Format | Placement | Image Spec | Video Spec | Text Limits |
|--------|-----------|-----------|-----------|-------------|
| Single Image | Feed, Stories, Reels | 1080×1080 (feed), 1080×1920 (stories) | N/A | Primary: 125 chars, Headline: 40, Description: 30 |

### Per-Ad Deliverables
- Primary text: 3 variations (short / medium / long)
- Headlines: 5 variations
- Descriptions: 3 variations
- CTA button: Learn More, Sign Up, Shop Now, Get Offer, Book Now, Download, Contact Us

## LinkedIn Ads

### Formats
- Sponsored Content (single image, video, carousel)
- Message Ads (InMail)
- Text Ads
- Conversation Ads
- Document Ads (PDF carousel)

### Character Limits
- Sponsored Content: Intro text 600 chars, Headline 200 chars
- Message Ads: Subject 60 chars, Body 1,500 chars
- Text Ads: Headline 25 chars, Description 75 chars

### LinkedIn-Specific Copy Angles
- Professional development: "Level up your [skill]"
- Industry insight: "[Industry] is changing. Here's how to stay ahead."
- ROI-focused: "Companies using [product] see [X]% improvement in [metric]"
- Peer pressure: "Your competitors are already using [approach]. Are you?"
- Thought leadership: "[Report] reveals [surprising finding]"

### Targeting Parameters to Recommend
- Job title (decision makers), seniority level
- Company size, industry
- Skills and interests
- Matched audiences (website retargeting, email lists)
- Lookalike audiences

## TikTok Ads

### Specs
- Format: In-Feed (video), TopView, Branded Hashtag Challenge, Spark Ads
- Video: 9:16 vertical, 5–60 seconds (9–15 seconds optimal)
- Resolution: 720×1280 minimum; file size up to 500 MB
- Ad text: 100 characters
- CTAs: Learn More, Shop Now, Sign Up, Download, Contact Us

### Creative Principles
- First 3 seconds determine watch rate — hook immediately
- Native aesthetic outperforms polished creative
- Use trending sounds; add text overlays for sound-off viewing
- Face-to-camera outperforms product-only shots
- Fast-paced jump cuts maintain retention

### 30-Second Script Template
```
[0-3s]  HOOK:     "Wait — you're still doing [old way]?"
[3-10s] PROBLEM:  Show the frustration / pain point visually
[10-20s] SOLUTION: Introduce product with quick demo
[20-25s] PROOF:    Flash testimonial, number, or result
[25-30s] CTA:      "Link in bio" or "Click to try free"
```

## Twitter/X Ads

### Formats
- Promoted Tweets (text, image, video, carousel)
- Follower Ads
- Amplify (video pre-roll)

### Specs
- Tweet text: 280 chars max (100–150 performs best)
- Image: 1200×675 or 1080×1080
- Video: up to 2:20, but 6–15 seconds optimal

### Copy Style
- Conversational, not corporate
- Hot take + solution format
- Thread-style: first tweet is the hook, rest is the story
